# 📚 Curso Angular - Projeto de Estudo

Projeto de aprendizado baseado em um curso de Angular no YouTube. Aqui estou praticando os conceitos fundamentais do framework Angular, desde componentes básicos até funcionalidades mais avançadas.

## 🎯 Sobre este projeto

Este é um projeto educacional onde estou aprendendo:
- ✅ Componentes e templates
- ✅ Data binding
- ✅ Serviços e injeção de dependência
- ✅ Roteamento
- ✅ Formulários
- ✅ HTTP e requisições
- E muito mais...

## 🚀 Como executar

### Servidor de desenvolvimento

Para iniciar o servidor de desenvolvimento:

```bash
ng serve
```

Navegue até `http://localhost:4200/`. A aplicação recarregará automaticamente quando você modificar os arquivos fonte.

### Gerar componentes

Para gerar um novo componente:

```bash
ng generate component nome-do-componente
```

### Build para produção

Para compilar o projeto:

```bash
ng build
```

Os artefatos serão armazenados no diretório `dist/`.

### Executar testes

Para rodar testes unitários com [Vitest](https://vitest.dev/):

```bash
ng test
```

### Testes e2e

Para testes end-to-end:

```bash
ng e2e
```

Angular CLI does not come with an end-to-end testing framework by default. You can choose one that suits your needs.
